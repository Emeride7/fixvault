Comment utiliser FixVault

Votre base de connaissances IT personnelle

01
Rechercher un problème

Utilisez la barre de recherche en haut pour chercher par titre, symptôme, commande ou tag. La recherche est instantanée.

02
Filtrer par catégorie

Cliquez sur une catégorie dans la colonne gauche (Windows, Network, Linux…) pour afficher uniquement les solutions correspondantes.

03
Consulter la solution

Cliquez sur une carte pour voir le détail complet : description du problème, étapes de résolution et commandes prêtes à l'emploi.

04
Copier les commandes

Chaque commande dispose d'un bouton Copy. Un clic suffit pour copier dans le presse-papiers.

05
Ajouter une solution

Vous avez résolu un problème non référencé ? Cliquez sur + Ajouter solution pour l'enregistrer pour la prochaine fois.

💡

Raccourci clavier : ⌘K ou Ctrl+K pour focuser la recherche instantanément.
